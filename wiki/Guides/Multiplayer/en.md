# Multiplayer
Play with other Rinari players in real time.

## Getting Started
To access multiplayer, click the **Multiplayer** button on the osu! main menu 
or press `F3`. Make sure you are connected to Rinari via 
[[Guides/Getting_Started|the devserver launch argument or Rinari client]].

## Creating a Room

1. Click **Create Room**
2. Set your room name and password (optional)
3. Select a beatmap
4. Configure your settings and wait for players to join
5. Click **Start Game** when ready

## Joining a Room

1. Browse the room list
2. Click on a room you want to join
3. Enter the password if required
4. Select your mods and click **Ready**
5. Wait for the host to start the game

## Room Settings

**Win Condition**
Determines how the winner is decided:
- **Score** — highest score wins
- **Accuracy** — highest accuracy wins
- **Combo** — highest combo wins
- **Score V2** — a combination of score, accuracy and combo

**Team Mode**
- **Head to Head** — every player competes individually
- **Tag Co-op** — players take turns hitting objects together
- **Team VS** — players are split into two teams, Red and Blue
- **Tag Team VS** — Tag Co-op but with teams

## Host
The host controls the room settings and can start the game. Host can be 
transferred to another player by clicking their name and selecting 
**Give Host**.

## Mods in Multiplayer
Each player can select their own mods. Some mods are only available if 
the host enables **Free Mods**. The following mods affect all players 
regardless of Free Mods:
- Double Time / Nightcore
- Half Time
- Hard Rock
- Easy
