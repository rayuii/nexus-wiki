# Rules
Welcome to Rinari! This is an anarchy server — do whatever you want, mostly.
There are only a few things we actually care about.

## The Only Rules

1. **One account per person.**
   No exceptions outside of approved bot accounts. Lost access? Contact us via the
   [Rinari Discord](https://discord.gg/fSmtpH3HSK) or [support@rinarii.de](mailto:support@rinarii.de).

2. **No DDoS or server abuse.**
   Deliberate attempts to overload, DDoS, or otherwise disrupt Rinari's servers
   are strictly prohibited. Permanent ban, no appeal.

3. **No Freedom or Maple.**
   These clients allow fake replay submission and score manipulation, which corrupts
   server data for everyone. Unlike other cheats, these actively harm the server 
   itself. Use of either will result in a restriction.

4. **Minimum age requirement.**
   All users must be at least 13 years old to use Rinari. Accounts found to be 
   under this age will be suspended until proof of eligibility is provided.

## Chat Rules

1. **Don't spam or flood.**
   Taking up half the screen by yourself is annoying. Don't do it.

2. **If someone gets silenced or restricted, leave it alone.**
   Their case is between them and the administrators. Don't make it a spectacle.

3. **Don't harass people after being blocked.**
   If a user blocks you, it means they no longer wish to communicate. Attempting 
   to bypass the block to harass, troll, or provoke them will result in a silence 
   or restriction.

## Forum Rules

1. **Don't spam posts in a thread.**
   Use the edit button. Excessive posting may lead to post removal or a punishment.

2. **Don't necro-post in threads older than 3 months.**
   If the topic is still relevant, start a new thread instead.

That's it. Genuinely.
